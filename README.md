# updated-object-analytics [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Dependency Status][daviddm-image]][daviddm-url]
> Notifies analytics services from updated objects

## Installation

```sh
$ npm install --save updated-object-analytics
```

## Usage

```js
var updatedObjectAnalytics = require('updated-object-analytics');

updatedObjectAnalytics('Rainbow');
```
